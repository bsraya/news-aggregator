# News Aggregator

1. First build the images
   ```
    make build
   ```
2. Run the server and the website
   ```
    make up
   ```
3. Go to the website
   ```
    http://localhost:3000
   ```

If you want to check what the API is receiving, go to `http://localhost:3000/get`. To terminate the server and the website, run `make down`.
